Flight Search

Sort of like Priceline or Kayak, but you get to choose the prices.

In this set of challenges, you are required to build a backend API that receives different query params and returns relevant results, suggesting flights for a potential user for a possible purchase. Each challenge adds features and possibilities upon the former one.

This set may assist with strenghthening API skills which are not plain RESTful ones. It also introduces gradually increasing complexity that will require performance tune-ups in order to acheive timely responses.
